streamMessage property

StreamMessageCallback streamMessage
read / write

接收到对方数据流消息的回调。

该回调表示本地用户收到了远端用户调用 RtcChannel.sendStreamMessage 方法发送的流消息。

StreamMessageCallback 包含如下参数：

  • int uid：用户 ID。
  • int streamId：数据流。
  • String data：接收到的数据。
