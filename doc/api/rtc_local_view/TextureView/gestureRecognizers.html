gestureRecognizers property

Set<Factory<OneSequenceGestureRecognizer>> gestureRecognizers
final, inherited

指定 Web 视图使用的手势。

如果存在多个手势识别器，其他手势识别器会与 Web 视图竞争指针事件的处权。例如，假设 Web 视图位于 ListView 中， ListView 会争取对垂直方向的拖动手势进行处理。Web 视图将声明已被其他手势识别器识别的手势。

如果该集合为空，Web 视图将仅处理没有被任何手势识别器声明的指针事件。
