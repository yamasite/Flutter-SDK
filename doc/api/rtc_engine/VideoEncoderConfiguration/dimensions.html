dimensions property

VideoDimensions dimensions
@JsonKey(includeIfNull: false), read / write

视频编码的分辨率 (px)，用于衡量编码质量，以长 × 宽表示，默认值为 640 × 360。 用户可以自行设置分辨率，也可以在如下列表中直接选择想要的分辨率：

  • 120x120
  • 160x120
  • 180x180
  • 240x180
  • 320x180
  • 240x240
  • 320x240
  • 424x240
  • 360x360
  • 480x360
  • 640x360
  • 480x480
  • 640x480
  • 840x480
  • 960x720
  • 1280x720

Note

  • 该值不代表最终视频输出的方向。请查阅 VideoOutputOrientationMode 了解设置视频方向。
  • 视频能否达到 720P 的分辨率取决于设备的性能，在性能配备较低的设备上有可能无法实现。如果采用 720P 分辨率而设备性能跟不上，则有可能出现帧率过低的情况。
